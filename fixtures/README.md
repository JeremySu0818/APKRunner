# Fixtures

This repository does not include third-party APKs.

Use only APKs you are legally allowed to inspect:

- Your own debug builds.
- Open-source APKs published by their authors for testing.
- APKs from internal test projects.

Avoid uploading proprietary APKs, production signing keys, user data, or apps obtained from unsafe mirror sites.

For Rust tests, the project builds tiny synthetic ZIP/APK fixtures in memory. Those fixtures are not runnable Android apps; they exist only to validate parser and compatibility behavior.
